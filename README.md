# Reading Tracker

A React + TypeScript + Tailwind web app for tracking books.

## Features

- Search for books using the Open Library API
- Add books to a personal library
- Prevent duplicate books from being added
- Move books through reading statuses:
  - To Read
  - Reading
  - Finished
- Rate finished books from 1 to 5 stars
- Delete books from the library
- Filter books by status
- Sort books by title, author, or date added
- View reading stats
- Save library data with localStorage
- Toggle light/dark mode

## Tech Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Open Library API
